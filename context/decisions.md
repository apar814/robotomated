# Robotomated — Decision Log

### 2026-03-20 — Foundation Decisions
| Decision | Choice | Why |
|----------|--------|-----|
| Brand name | Robotomated (past tense = job done) | "Your warehouse, robotomated." Works for every customer type. Conversation starter. |
| Domain | robotomated.com primary, robotomate.com redirects, robotomato.com parked | Three domains cover all name variants. Primary is outcome-focused. |
| Build approach | Content-first, not product-first | Content compounds. SEO lead cannot be bought back. 12-month head start > any feature. |
| Phase 1 focus | Reviews + "Best Of" guides + AI advisor | These three alone generate affiliate revenue and build trust. Everything else is Phase 2+. |
| Stack | Next.js 15 + Supabase + Vercel (all free tier) | $0/mo infrastructure. 30+ months runway on $10K. Don't over-engineer. |
| AI | Claude API (Sonnet) for advisor | Best reasoning for complex multi-turn advisory conversations. Provider-agnostic wrapper for future flexibility. |
| Scoring | RoboScore — 8 dimensions, public methodology | Wirecutter model. Transparent, explainable, category-weighted. THE moat. |
| Editorial | Independence non-negotiable | Manufacturers cannot influence scores. Period. This is the entire business. |
| Monetization sequence | Affiliate → Pro → Marketplace → Enterprise | Don't monetize before trust. Affiliate is zero-risk first revenue. |
| SEO | Programmatic pages for every robot + "Best Of" per category | 10,000+ indexable pages. Each one is a trust signal + traffic source. |
| Budget | $10K total, ~$200-300/mo burn | 30+ months runway. No VC needed until $100K MRR proves model. |
| No marketplace Phase 1 | Content + AI first, marketplace later | Marketplace requires liquidity (buyers AND sellers). Build audience first, marketplace has customers ready. |
| Simple architecture | No monorepo Phase 1 | Single Next.js app. Add Turborepo when marketplace/admin justify the complexity. |
| Seed data | 200 robots across 10 categories | Enough to demonstrate value and test all features. Scale to 2,000 by Month 6. |
