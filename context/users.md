# Robotomated — Users

## Seven Personas

### Curious Consumer (PRIMARY — Phase 1 focus)
Homeowner, 35-55, moderate tech comfort, budget-conscious.
Need: "What robot will actually help me at home?"
Journey: Google → "best home robot 2026" → lands on guide →
  AI advisor asks 3-5 questions → 3 recommendations → reads review →
  clicks "Best Price" → affiliate link. Done.

### Tech Enthusiast
Early adopter, 25-40, high tech comfort, wants depth and specs.
Need: deep reviews, benchmarks, community, early access.
Monetization: Pro subscriber ($49/mo), marketplace buyer.

### Parent / Educator
Parent or K-12 teacher, wants educational robotics for kids.
Need: curriculum alignment, age-appropriateness, project ideas.
Content: RoboLearn education track.

### SMB Owner
Restaurant, retail, warehouse — evaluating labor-saving robots.
Need: ROI data, vendor comparison, case studies, pricing transparency.
Monetization: marketplace buyer, Pro subscriber.

### Enterprise Buyer
VP Operations/IT, Fortune 1000, multi-unit procurement.
Need: fleet management, compliance, integration, TCO analysis.
Monetization: Enterprise SaaS ($2,499+/mo). Phase 3.

### Developer / Maker
Software engineer or hobbyist building custom projects.
Need: API docs, SDK access, component marketplace, community code.
Monetization: parts marketplace, Pro subscriber.

### Researcher / Academic
University lab, think tank, policy analyst.
Need: structured data access, market intelligence, citation-quality.
Monetization: market intelligence reports ($2.5K-50K).

## Design Principle
Complexity on demand. Simple default view, deep data available on click.
AI assists, never gatekeeps — users can always browse and search
without talking to AI.

## Key User Journeys

### Journey 1: First-Time Robot Buyer (80% of Phase 1 traffic)
1. Arrives via Google ("best home robot 2026")
2. Sees "Start Here" — picks goal from visual use-case grid
3. AI Advisor asks 3-5 questions → top 3 recommendations
4. Reviews each: RoboScore breakdown, video, expert review
5. Clicks "Best Price" → marketplace or affiliate link
6. Optionally creates account to save comparison, set price alert

### Journey 2: Enterprise Procurement Officer (Phase 3)
1. Lands from LinkedIn ad or Google search → Enterprise page
2. Books demo OR self-serves with 14-day trial
3. Uploads current robot fleet → AI maps to database, surfaces gaps
4. Runs vendor assessment for 3 competing platforms
5. Downloads 40-page AI-generated procurement report
6. Signs Enterprise at $2,499/month
