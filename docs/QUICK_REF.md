robotomated.com (primary) | robotomate.com (redirect) | robotomato.com (parked)

## Stack

Zustand + TanStack Query + Recharts + Resend + PostHog + Sentry

Phase 3 (P10-15): affiliate, data pipeline, Stripe, analytics, launch

## Critical Rules
- Editorial independence is THE moat. Never compromise.
- Content first, monetization second. Trust before revenue.
- SSR everything — SEO is how we win Google.
- AI assists, never gatekeeps. Browsing always works without AI.
- Mobile-native. 60%+ discovery is mobile.

Write for the foreman, not the PhD. Don't dumb it down for either.
